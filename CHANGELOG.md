## [1.7.0] — 2026‑02‑21

### Added
- Initial public release of LogSmith
- SmartLogger core class
- Structured formatting system
- Color and gradient output engine
- Rotation and retention logic
- Global auditing mode
- Dynamic log levels
- Themes and palette blending
- Thread‑safe and process‑safe file handlers
- Full documentation set
- Examples and developer tools included in the wheel
